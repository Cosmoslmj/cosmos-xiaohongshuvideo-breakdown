# Cosmos Xiaohongshu Video Breakdown

宇宙小红书爆款分析 skill。用于拆解小红书/短视频爆款链接、视频文案、截图或本地视频文件，输出爆款原因、可迁移结构、延伸选题、完整口播文案、封面标题、拍摄画面建议和发布后复盘。

## 触发方式

可以这样调用：

```text
用 宇宙爆款视频分析skill 分析这个爆款视频：{链接/文案/截图/视频}
```

也可以使用：

```text
用 宇宙小红书爆款分析skill 分析这个爆款视频：{...}
```

或英文/文件名：

```text
用 cosmos-xiaohongshuvideo-breakdown 分析这个爆款视频：{...}
```

## 能做什么

- 分析爆款视频为什么火
- 拆选题、前 3 秒钩子、内容推进、封面标题、收藏理由
- 给出内容诊断、爆款评分表和可迁移骨架
- 推荐适合用户的 3 个延伸选题
- 生成可直接拍的口播文案，并标注字数和建议时长
- 在文案中标注适合插入的截图、录屏和出镜画面
- 输出封面标题、拍摄执行表、可领取资产建议和发布后复盘标准

## 安装

把本仓库目录放进支持 Skills 的 Agent 的 skills 目录中，确保 `SKILL.md` 位于 skill 根目录。

例如：

```text
skills/cosmos-xiaohongshuvideo-breakdown/SKILL.md
```
